# iotsr
IoT con Arduino Nano , LCD 5110 , ESP8266 WIFI  configurable desde PC por USB

Hardware Usado

1.- Arduino Nano Compatible Cedido por David.

https://www.arduino.cc/en/Main/ArduinoBoardNano

Se puede comprar en ebay por menos de 4€

2.- Fuente de alimentación y 65pcs MB102 jumper kit protoboard cable y 400 agujeros

Desde 6€ en banggood

3.- ESP8266-01 WIFI Module

4.- LCD 5110
